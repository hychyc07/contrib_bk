Kinect SDK 1.5 is required to compile it.
In app/scripts there is a script to run everything properly. It is needed to launch the actionRad.rad file contained in RAD folder in order to
send "start", "win", "lose" and "stop" commands. If it is wanted to not use RAD, it is possible to send the same commands at the port
/demoActionRecognition/rpc.
In app/conf there is an image containing examples of the supported actions.
